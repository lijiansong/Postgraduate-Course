## vim taglist plugin
```
set nu                                                                                                                                                                                                            
set syntax=on
set cindent
set expandtab
set tabstop=4
set shiftwidth=4
set hlsearch
set nocompatible
set ignorecase
set incsearch
set cursorline
set encoding=utf-8
"set fileencoding=ucs-bom,utf-8,cp936,big5,euc-jp,euc-kr,latin1
filetype on
filetype plugin on
filetype indent on
map <silent> t :TlistToggle<cr>
```
